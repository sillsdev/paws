Using the PAWS-generated PC-PATR grammar file in FieldWorks Language Explorer (aka FLEx)

To use the PAWS-generated PC-PATR grammar file with a FLEx language project, do the following:

  • Initial set-up
    1. Exit out of FLEx.
    2. Read the User Documentation found here.
    3. Download and install the Use PC-PATR with FLEx program from here.
    4. Run the FLEx program. Use the Tools menu item / Utilities.... Find the "Use PC-PATR with FLEx" item, check it, and then click on the “Run Checked Utilities Now” button.
    5. (The first time you do this, it will add the list of feature descriptors needed as well as create a custom field in senses called PCPATR.)
    6. To see the User Documentation: click on the "Help" button and choose "User Documentation."
  • Adding feature names in FLEx
    1. Use Lexicon / Bulk Edit Entries / List Choice.
    2. Show the custom PCPATR field.
    3. Set the Target field to PCPATR.
    4. Use filters to find the appropriate entries.
    5. Click on the "Choose" button under "Change to:" and select all the feature names needed. (Please note that if you've just added some feature names, this list will still have them selected; you will need to unselect them first.)
    6. Use the "Preview" and "Apply" buttons as normal to set the feature names for the selected entries.
  • Category mapping
    1. The various "Category Mapping" pages talk about what category you use. The one used by the Use PC-PATR with FLEx program will be the category abbreviation in the default analysis writing system in FLEx. If this does not match what PAWS uses, enter the FLEx value in the box of the category mapper.
  • Processing the test files:
    1. Import each test file into FLEx using File / Import / Standard Format Interlinear.
    2. Analyze the words in the text(s) by using one of the FLEx parsers or by hand. Be sure to include a word category (these will necessarily be shown using FLEx's values).
    3. In the Use PC-PATR with FLEx window, use the Browse button to select the PC-PATR grammar file.
    4. Use the "Disambiguate & show this text" button or the "Parse & show this segment" button to test the grammar on the text.
    5. Examine the results in the PcPatrBrowser tool (which comes up automatically).
