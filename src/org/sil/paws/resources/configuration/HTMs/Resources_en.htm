PAWS Starter Kit Resources

The following resource files are available for helping you understand various aspects of the PAWS Starter Kit.

The first four relate to the "writer" aspect of PAWS.

  • XLingPaper Reference Guide for Editing PAWS Starter Kit Output using the XMLmind XML Editor. This lists a number of common tasks users often want to do when editing the XLingPaper output from the PAWS Starter Kit.

  • XLingPaper Quick Reference Guide for Editing PAWS Starter Kit Output using the XMLmind XML Editor. A shorter version of the reference guide, more suitable for printing out.

  • Structure of an Interlinear Example in XLingPaper. This provides a diagram showing the detailed structure of an interlinear example.

  • XLingPaper's main web page. This discusses the nature of the writer format and includes a reference to some tools that make it relatively easy to edit the writer output.

The last three relate to the PC-PATR parser and the Government and Binding theory of Syntax. (Please note that PAWS does not overtly follow GB syntax. It does make use of X-Bar structure though.)

  • A discussion of using PC-PATR to model GB Syntax. This paper reports the results of an experiment in modeling English syntax with PC-PATR and a GB model of syntax.

  • An introduction to GB Syntax This provides a gentle, step-by-step introduction to the Government and Binding theory of syntax.

  • A reference manual for PC-PATR This is a detailed reference guide to the PC-PATR syntactic parsing program.
